## Project 04: Theme Classification & Root Cause Analysis

### Overview
This phase of the project focused on identifying and refining recurring themes within employee voice data using a combination of AI-assisted analysis, operational questioning, and Python-based categorization.

Building on previous sentiment analysis work, this stage moved beyond measuring sentiment alone and focused more directly on uncovering patterns connected to attrition and employee experience.

### Objective
To identify recurring operational and workplace themes within employee feedback, classify those themes across reviews, and explore the root causes behind one selected issue area.

### What Was Done
- Used AI tools to identify recurring themes within employee voice data
- Refined and validated AI-generated themes using analytical questioning and previous findings
- Used Python to create a theme classification column within the dataset
- Tagged reviews according to 1 of 3 identified themes
- Conducted sentiment analysis across categorized themes
- Created visualizations to compare sentiment patterns between themes
- Selected one theme for deeper operational analysis
- Used Claude AI to generate a root cause tree visualization to map contributing factors and downstream impact

### Tools Used
- Python
- Pandas
- TextBlob
- AI-assisted thematic analysis
- Claude AI (root cause visualization)

### Key Questions Explored
- What recurring themes consistently appear across employee reviews?
- How can AI-assisted analysis support pattern recognition within unstructured data?
- How do sentiment trends differ across categorized themes?
- What operational or structural factors contribute to the selected issue area?

### Key Insights
AI-assisted analysis helped accelerate the process of identifying recurring themes, but the refinement process still required contextual understanding and analytical judgment.

Using insight-driven questioning alongside AI-generated themes helped improve the accuracy and relevance of classifications.

After categorizing reviews into themes and analyzing associated sentiment, clear patterns began to emerge between employee experience and operational support structures.

Exploring one theme more deeply through root cause analysis revealed how operational pressure, leadership gaps, and support systems can interact to influence employee experience and attrition.

### Deliverables

- Theme sentiment visualizations (see `/visuals`)
- Root cause tree analysis (see `/analysis`)
